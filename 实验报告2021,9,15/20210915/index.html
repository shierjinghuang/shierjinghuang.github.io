# 实验二：虽然我也不知道我当时在干什么

今天上午的实验课，由于去年我已经安装和自学过 python 语言，完成课上的内容并没有花费很多时间，在完成课上的任务之外，我大部分的时间都在尝试安装并配置 jupyter notebook 的 python 环境。

配置方案在 CSDN 里面我找到了这个配置的方案，但是在学校机房的尝试却没能成功。在我配置环境时 cmd 系统报错，提示环境变量的 path 的配置存在问题，在我将 jupyter notebook 对应的地址加入 path 变量时，cmd 又报了另一个错误。由于没有保留截图，这里也无法展示，在后来请教老师也没有得出结论，后来回到宿舍在自己和舍友电脑上配置 jupyter notebook 想要复现当时出现问题，然而并没有出现什么问题。

最后怀疑是机房电脑是 win7 操作系统的原因，jupyter notebook 可能没有兼容较低的 windows 版本。

不过整个实验课，我确实也是重新自己配置了一遍 jupyter 的在线编程环境，也是有所收获。
